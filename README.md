# 🚌 Hyderabad City Bus Dashboard

A modern GIS dashboard built using

- OpenStreetMap
- Leaflet
- Chart.js
- SheetJS
- DataTables

## Features

✔ Reads Excel directly

✔ Interactive Map

✔ Light / Dark Mode

✔ Dashboard Cards

✔ Charts

✔ Search

✔ Filters

✔ Depot Statistics

✔ Mobile Responsive
